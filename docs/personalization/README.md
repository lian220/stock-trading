## 개인화 기능 개요

현재 시스템은 전역 추천 및 자동매매 설정만 지원하고 있습니다. 개인화 시스템을 통해 다음과 같은 기능을 제공합니다:

### 주요 기능

1. **사용자 프로필 관리**
   - 리스크 프로필 (conservative, moderate, aggressive)
   - 투자 스타일 (growth, value, dividend, mixed)
   - 선호 섹터/산업 설정
   - 투자 기간 및 금액 범위 설정

2. **개인화된 추천 시스템**
   - 사용자별 활성 종목만 추천
   - 사용자별 가중치 적용 (AI/기술적/감정 분석)
   - 선호 섹터/산업 기반 점수 조정
   - 과거 거래 이력 기반 학습

3. **개인화된 자동매매 전략**
   - 리스크 프로필에 따른 손절/익절 기준 자동 조정
   - 사용자별 최소/최대 투자 금액 적용
   - 선호 섹터 우선순위 반영

4. **거래 이력 추적 및 학습**
   - 사용자별 거래 이력 저장
   - 성공률 기반 종목 필터링
   - 거래 패턴 분석
