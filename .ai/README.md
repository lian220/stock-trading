# AI 에이전트 공통 규칙 디렉토리

이 디렉토리는 모든 AI 에이전트 툴(커서, 클로드, 안티그래비티 등)이 공통으로 참조하는 규칙 파일들을 포함합니다.

## 📁 디렉토리 구조

```
.ai/
├── README.md                    # 이 파일
├── PROJECT_RULES.md             # 프로젝트 공통 규칙 및 체크리스트
├── rules.md                     # 프로젝트 상세 규칙 및 가이드라인
├── conventions/                 # 컨벤션 파일들
│   ├── COMMIT_CONVENTION.md     # 커밋 메시지 컨벤션
│   └── CODING_CONVENTION.md     # Python 코딩 스타일 가이드
├── guides/                      # Agent 가이드 파일들
│   ├── TEST_GUIDE.md            # 테스트 코드 생성 가이드
│   ├── REFACTORING_GUIDE.md     # 리팩토링 가이드
│   └── PROMPT_GUIDE.md          # 프롬프트 가이드
└── ...
```

## 🔗 AI 에이전트 툴별 설정 파일

각 AI 에이전트 툴은 다음 설정 파일을 통해 이 공통 규칙을 참조합니다:

### Cursor
- **설정 파일**: `.cursor/rules/*.mdc`
- **레거시**: `.cursorrules`
- **참조**: 이 디렉토리의 모든 공통 규칙 파일

### Claude Code
- **설정 파일**: `.claude/CLAUDE.md`
- **참조**: 이 디렉토리의 모든 공통 규칙 파일

### Antigravity
- **설정 파일**: `antigravity.md` (프로젝트 루트)
- **참조**: 이 디렉토리의 모든 공통 규칙 파일

## 📝 규칙 파일 수정 시 주의사항

1. **공통 규칙 수정**: `.ai/` 디렉토리의 파일을 수정하면 모든 툴에 적용됩니다.
2. **툴별 설정 수정**: 각 툴의 설정 파일(`.cursor/rules/`, `.claude/`, `antigravity.md`)은 공통 규칙을 참조하도록 유지해야 합니다.
3. **일관성 유지**: 모든 툴이 동일한 규칙을 따르도록 공통 규칙 파일을 우선적으로 사용하세요.

## 🎯 규칙 파일 우선순위

1. **공통 규칙** (`.ai/` 디렉토리) - 모든 툴이 참조
2. **툴별 설정** (`.cursor/rules/`, `.claude/`, `antigravity.md`) - 각 툴 전용 설정
3. **레거시 파일** (`.cursorrules`) - 참고용
